# JB's Vainas — PWA

App instalable en el celular (Android/iOS) directo desde el navegador, sin App Store.

## Archivos (van todos en la raíz del repositorio)
- `index.html` — la app
- `manifest.json` — la "cédula" de la PWA (nombre, colores, íconos)
- `service-worker.js` — permite que abra aunque no tengas internet
- `icons/` — tu foto en 4 tamaños (32, 180, 192, 512 px)

## Pasos en GitHub
1. Crea un repositorio nuevo, público, sin README (para no chocar).
2. Sube estos 4 elementos tal cual (arrastra la carpeta completa).
3. Ve a **Settings → Pages → Branch: main → carpeta /(root) → Save**.
4. Espera 1-2 minutos. Te da un link tipo `https://tu-usuario.github.io/nombre-repo/`.

## Instalar en el celular
- **Android (Chrome):** abre el link → menú (⋮) → "Instalar app" / "Añadir a pantalla de inicio".
- **iPhone (Safari):** abre el link → botón compartir (⬆️) → "Añadir a pantalla de inicio".

Listo, queda como un ícono más en tu pantalla, pantalla completa, sin barra del navegador.
